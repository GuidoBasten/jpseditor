JPSeditor is a graphical user interface to create the geometry of a scenario simulated by JPScore. It comes with set of CAD- and further tools to simplify the creation of proper xml-files incorporating information about the scenario geometry.

This software is platform independent, released under the LGPL license and is written in C++ with Qt.
